Reconstruction by K. Dahan

Web page by Michael Gogins

Original version by Victor Lazzarini

This Web page provides an example of the simplest possible Web page that plays a Csound piece, embedded in the page and editable, using the PNaCl build of Csound. All unnecessary HTML elements and JavaScript function calls have been omitted. To see the code behind this page, right-click on the page and select "view page source".

Comments in the page source explain what each element is for and why it is written the way it is.
