Web page by Michael Gogins

Original version by Victor Lazzarini

This Web page provides an example of the simplest possible Web page that plays a Csound piece, embedded in the page and editable, using the PNaCl build of Csound. All unnecessary HTML elements and JavaScript function calls have been omitted.

Comments in the page source explain what each element is for and why it is written the way it is. You can view the code and comments by right-clicking on this page and selecting "View page source".
